# Jwt Toolkit

JWT manipulation toolkit.

Toolkit manipulasi JWT.


## Description

Provides tools for decoding, analyzing, verifying, and generating JSON Web Tokens.

## Deskripsi

Menyediakan alat untuk mendekode, menganalisis, memverifikasi, dan menghasilkan JSON Web Token.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Decode: python jwt_tool.py decode "eyJhbGciOiJ..."
- Verify: python jwt_tool.py verify "token" "secret"
- Demo: python jwt_tool.py --demo

## Penggunaan

- Decode: python jwt_tool.py decode "eyJhbGciOiJ..."
- Verify: python jwt_tool.py verify "token" "secret"
- Demo: python jwt_tool.py --demo
